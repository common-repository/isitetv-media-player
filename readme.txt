Allows the use of the shortcode "isitetv_player" to insert an iSiteTV media player into your page.
 
== Description ==

An easy way to include the media player from iSiteTV.

This plugin is nice and simple, just install it, activate it and insert the shortcode "isitetv_player" on your page.

For the shortcode to work you are required to include an encoded value - this is provided by our Content team.

An exampe of the shortcode is:
[isitetv_player encoded="en012345678901234567890123456789"]
 
== Installation ==
 
To manually install this plugin, all you need to do is:
1. Upload `isitetv_media_player.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
